## 🚀 Getting Started (5 Minutes)

```bash
# 1. Setup
cd qa_testing_agent
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 2. Configure
cp .env.example .env
# Add: OPENAI_API_KEY=sk-...

# 3. Create Sample CSV
python main.py
# Select: Option 3 (Create Sample CSV)

# 4. Edit Sample CSV
# Open ./stories/sample_stories.csv
# Modify with your stories

# 5. Run System
python main.py
# Select: Option 1 (CSV Stories)
# Press Enter (uses ./stories)

# 6. Make Approval Decision
# Select option 1 (Approve All)

# 7. Get Report
# HTML report in ./reports/
```
