# 🎉 QA Testing Agent v3.0+ - START HERE

**Welcome!** You have received a **complete, production-ready QA automation system**.

---

## ⚡ 30-Second Overview

A **platform-agnostic test automation system** that:

1. **Imports stories from CSV** (any system - Azure DevOps, Jira, Linear, etc.)
2. **Generates test cases** using AI
3. **Requires manual approval** before execution
4. **Executes tests** with Selenium/Playwright
5. **Generates professional reports** with metrics

**No API credentials. No platform lock-in. Just CSV files.**

---

## 🚀 Quick Start (5 Minutes)

```bash
# 1. Setup (2 min)
cd qa_testing_agent
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 2. Configure (1 min)
cp .env.example .env
# Edit: Add OPENAI_API_KEY=sk-...

# 3. Run (2 min)
python main.py
# Select: 1 (CSV Stories)
# Select: 3 (Create Sample CSV) on first run
# Edit: ./stories/sample_stories.csv
# Run again: Select 1, approve all

# 4. Results
# Check: ./reports/qa_suite_*.html
```
